Module contents¶

Bumps: curve fitter with uncertainty estimation

This package provides tools for modeling parametric systems in a Bayesian context, with routines for finding the maximum likelihood and the posterior probability density function.

A graphical interface allows direct manipulation of the model parameters.

See https://bumps.readthedocs.io for online manuals.
